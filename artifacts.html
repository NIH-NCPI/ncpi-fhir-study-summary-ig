This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide

Harmonization Map

A ConceptMap which captures the mappings from the study’s variable names to their public ontological mappings
NCPI Study DD Data Table

Aggregates Variable Details associated with a single dataset table that are represented in FHIR
NCPI Study DD Data Variable

Describes a single variable from the dataset
Study Summary

An observation containing summary data associated with research data from FHIR

The code MUST indicate that it is a summary and the subject MUST point to the group of participants if the summary doesn’t pertain to the entire study.

The StudySummary MUST reference a study in the focus.
NCPI Summary Group

Represent a group of research subjects, either the entire study’s population or a subgroup such as an individual consent group
NCPI Summary Study

Represent a research study within the context of summary/metadata.
Variable Summary

An observation containing summary data associated with research data from FHIR

The code MUST indicate that it is a summary and the subject MUST point to the group of participants to which the observation is based on.
